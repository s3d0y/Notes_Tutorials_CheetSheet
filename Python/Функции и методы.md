Метод - это функция, которая принимает экземпляр класса в качестве своего первого параметра. Методы являются членами классов.

```Python
t = 'Привет! Этот тёкст я написал моими руками'
t.capitalize()


#у переменной после точки идет вызов метода функции
# искать табом. 
# далее если поставить пустые кобки будет подсказка
# shift + tab - слает подсказку подробнее

# Методы функции
capitalize #делает первую букву большой, остальные маленькие
replace(old, new) #возвращает копию с заменой символа
strip #возвращает копию строки, удаляя как начальные, так и конечные символы (табы пробелы).
lower #перевод в нижний регистр
center #размещает строку по центру, справа и слева дополняя ее до указанной длинны
split #разбивает строку на подстроки (по умолчанию разелители это вайтспэйсы. Но вроде как можно и поменять)
join #соединяет строки, используя somestring как разделитель
append #добавляет в конец списка элемент, переданный ему в качестве аргумента.
extend #принимает в качестве параметра итерируемый объект (cписок, кортеж или строку)и объединяет его со списком.
```
