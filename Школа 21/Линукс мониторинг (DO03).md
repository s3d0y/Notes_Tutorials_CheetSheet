$# - количество параметров скрипта

``` sh
if [[ "$1" =~ ^-?([0-9]+\.?[0-9]*|\.[0-9]+)$ ]]; then
	echo "Error. Need text enter. Try again."
	exit 1
fi
```

$1 - первый параметр скрипта
=~ оператор сравнения с регулярным выражением 
`^[0-9]+$` - регулярное выржение: 
- ^ начало строки 
- `[0-9] +` -  одна или более цифр
- $ конец строки
